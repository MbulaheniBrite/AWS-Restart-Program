## ğŸ’¡ What is Amazon EC2?

Amazon EC2 is designed to make web-scale cloud computing easier for developers.

* **Ease of Use:** Its simple web service interface allows you to obtain and configure capacity with minimal friction.
* **Complete Control:** It provides you with complete control of your computing resources, letting you run on Amazon's proven computing environment.
* **Agility & Speed:** It reduces the time required to obtain and boot new server instances to minutes, allowing you to quickly **scale capacity, both up and down**, as computing requirements change.
* **Economic Model:** Amazon EC2 changes the economics of computing by allowing you to **pay only for capacity that you actually use**.
* **Resilience:** It provides developers the tools to build **failure resilient applications** and isolate themselves from common failure scenarios.
